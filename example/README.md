# Examples

## BasicC

BasicC is a C project. Run the following command to build and run the main
executable.

```
$ btool -root example/BasicC -target main -run
```

## BasicCC

BasicCC is a C++ project. Run the following command to build and run the main
executable.

```
$ btool -root example/BasicCC -target main -run
```

Run the following command to build and run a test.

```
$ btool -root example/BasicCC -target dep-1/dep-1-test -run
```
